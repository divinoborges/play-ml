## Why

The project references an MIT License in the README but has no LICENSE file, and the README shields are static badges with placeholder `#` links instead of dynamic shields pointing to the actual repo.

## What Changes

- **Add LICENSE file** — Create an MIT license file at the project root with the correct year and author
- **Fix README shields** — Replace static badge URLs with dynamic shields.io badges that link to the actual GitHub repo (build status from GitHub Actions, license from repo, demo link to GitHub Pages)
